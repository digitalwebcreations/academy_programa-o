Aula 7 - Orientação a Objetos

A programação orientada a objetos (POO) é um paradigma que organiza o código em objetos, tornando programas grandes mais fáceis de entender e manter.

Conceitos principais

  • Classe: modelo que define atributos e comportamentos de um objeto.
  • Objeto: instância de uma classe, com valores específicos.
  • Encapsulamento: proteger dados internos e fornecer acesso controlado.
  • Herança: criar classes base e estender funcionalidades em classes filhas.
  • Polimorfismo: usar métodos diferentes com mesmo nome dependendo do objeto.

Exemplo simples

class Pessoa:
    def __init__(self, nome, idade):
        self.nome = nome
        self.idade = idade

p1 = Pessoa("Hayan", 14)
mostrar p1.nome  # Hayan
mostrar p1.idade # 14

Vantagens da POO

  • Facilita a manutenção e evolução do sistema.
  • Permite reutilizar código de forma eficiente.
  • Organiza melhor programas complexos, separando responsabilidades.
