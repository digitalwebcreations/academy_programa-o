Aula 3 - Estruturas Condicionais

Nesta aula, vamos aprender sobre estruturas condicionais, que permitem ao programa tomar decisões com base em condições específicas.

O que são estruturas condicionais?

Estruturas condicionais permitem executar diferentes trechos de código dependendo de certas condições. A mais comum é o if/else.

Exemplo simples

idade = 18
if idade >= 18:
    mostrar "Você é maior de idade"
else:
    mostrar "Você é menor de idade"

O programa verifica a idade e mostra a mensagem correspondente.

Tipos de estruturas condicionais

  • if: verifica uma condição.
  • else: executa caso nenhuma condição seja verdadeira.
  • elif / else if: verifica condições adicionais.
  • switch / case: usado em algumas linguagens para comparar valores específicos.

Operadores comuns

Para criar condições, usamos operadores:

  • == : igual a
  • != : diferente de
  • > : maior que
  • < : menor que
  • >= : maior ou igual a
  • <= : menor ou igual a
  • and, or, not : operadores lógicos

Exemplo prático

nota = 7
if nota >= 7:
    mostrar "Aprovado"
elif nota >= 5:
    mostrar "Recuperação"
else:
    mostrar "Reprovado"

Dicas importantes

  • Use indentação correta (espaços ou tabs) para organizar seu código.
  • Planeje a lógica antes de codificar, evitando condições redundantes.
  • Teste todas as possibilidades para garantir que o programa se comporte como esperado.
