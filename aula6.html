Aula 6 - Arrays e Listas

Nesta aula, vamos aprender sobre arrays e listas, estruturas que permitem armazenar múltiplos valores de forma organizada.

O que são arrays e listas?

Arrays e listas são coleções de valores que podem ser acessados por índices, ou seja, cada elemento possui uma posição específica.

Exemplo de lista

numeros = [1, 2, 3, 4, 5]
nomes = ["Hayan", "Maria", "João"]

Acessando elementos

Para acessar valores, usamos o índice (começando em 0):

mostrar numeros[0]  # Mostra 1
mostrar nomes[2]    # Mostra "João"

Adicionando e removendo elementos

  • Adicionar: `numeros.append(6)`
  • Remover: `numeros.remove(3)`
  • Alterar: `numeros[1] = 10`

Laços com listas

Podemos percorrer uma lista usando loops:

for numero in numeros:
    mostrar numero

Dicas importantes

  • Escolha o tipo de coleção adequado (array, lista, dicionário) para cada situação.
  • Mantenha o código limpo e evite modificar listas enquanto percorre, para não gerar erros.
  • Pratique criando listas de diferentes tipos de dados e manipulando os elementos.
