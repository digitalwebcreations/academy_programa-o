Aula 2 - Variáveis e Tipos de Dados

Nesta aula, vamos explorar variáveis e tipos de dados, conceitos fundamentais que permitem armazenar e manipular informações em qualquer programa.

O que são variáveis?

Variáveis são espaços na memória do computador que armazenam valores temporariamente. Pense nelas como caixas com etiquetas que guardam informações que podem mudar ao longo do tempo.

Exemplo de variável

idade = 14
nome = "Hayan"
altura = 1.65

Aqui, criamos três variáveis: idade (número), nome (texto) e altura (número decimal).

Tipos de dados mais comuns

  • Números inteiros (int): 0, 1, 100, -5
  • Números decimais (float/double): 3.14, 1.65
  • Texto (string): "Olá mundo", "Hayan"
  • Booleano (bool): verdadeiro/falso, usado em decisões do programa
  • Listas/arrays: coleções de valores armazenados em sequência

Como criar variáveis corretamente

Algumas boas práticas:

  • Use nomes significativos: ex: `notaAluno` em vez de `x`.
  • Evite caracteres especiais e espaços.
  • Escolha o tipo de dado correto para cada informação.

Exemplo prático

Vamos calcular a média de notas de um aluno:

nota1 = 8
nota2 = 7
nota3 = 9
media = (nota1 + nota2 + nota3) / 3
Mostrar "A média do aluno é:", media

Dicas importantes

  • Variáveis podem mudar durante o programa, por isso o nome deve refletir seu propósito.
  • Tipos de dados ajudam o computador a entender como tratar cada informação.
  • Pratique criando variáveis diferentes e manipulando valores.

Compreender variáveis e tipos de dados é essencial para seguir em frente. Na próxima aula, aprenderemos sobre estruturas condicionais que permitem ao programa tomar decisões com base nessas informações.
