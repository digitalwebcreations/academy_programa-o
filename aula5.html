Aula 5 - Funções e Procedimentos

Funções e procedimentos são blocos de código que executam tarefas específicas, tornando o programa mais organizado e reutilizável.

O que são funções?

Funções recebem dados de entrada, processam e podem retornar um resultado. Elas permitem reaproveitar código e facilitar manutenção.

Exemplo de função

def soma(a, b):
    return a + b

resultado = soma(5, 10)
mostrar resultado

O que são procedimentos?

Procedimentos são semelhantes a funções, mas não retornam valores. Eles realizam ações, como imprimir informações ou modificar dados.

Exemplo de procedimento

def saudacao(nome):
    mostrar "Olá,", nome

saudacao("Hayan")

Benefícios de usar funções e procedimentos

  • Reutilização de código.
  • Melhor organização e clareza.
  • Facilidade para testar e corrigir erros.
