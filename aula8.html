Aula 8 - Manipulação de Arquivos

Manipular arquivos é essencial para armazenar dados permanentemente e acessar informações posteriormente.

Tipos de arquivos

  • Texto (.txt, .csv)
  • Planilhas (.xlsx)
  • Banco de dados (.db, .sql)
  • JSON (.json)

Exemplo de leitura e escrita

# Escrevendo em um arquivo
arquivo = open("dados.txt", "w")
arquivo.write("Olá, mundo!")
arquivo.close()

# Lendo um arquivo
arquivo = open("dados.txt", "r")
conteudo = arquivo.read()
mostrar conteudo
arquivo.close()

Boas práticas

  • Fechar sempre os arquivos após o uso.
  • Usar blocos try/catch para tratar erros.
  • Evitar sobrescrever arquivos importantes sem backup.
