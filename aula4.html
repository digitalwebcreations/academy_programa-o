Aula 4 - Laços de Repetição

Os laços de repetição são usados para executar blocos de código várias vezes, economizando tempo e evitando repetições desnecessárias.

Tipos de laços

  • for: repete um bloco de código um número definido de vezes.
  • while: repete enquanto uma condição for verdadeira.
  • do/while: similar ao while, mas executa pelo menos uma vez antes de verificar a condição.

Exemplo com for

for i in range(1, 6):
    mostrar i

O código imprime os números de 1 a 5.

Exemplo com while

contador = 1
while contador <= 5:
    mostrar contador
    contador = contador + 1

Aplicações práticas

  • Imprimir listas de informações.
  • Processar dados de forma automática.
  • Executar tarefas repetitivas sem duplicar código.

Dicas importantes

  • Evite loops infinitos garantindo que a condição eventualmente se torne falsa.
  • Use loops combinados com estruturas condicionais para maior flexibilidade.
  • Loops aninhados permitem trabalhar com tabelas e matrizes.
