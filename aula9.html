Aula 9 - Projeto Final de Programação

Nesta aula, você aplicará todos os conceitos aprendidos no curso de programação criando um projeto completo.

Objetivo do projeto

Desenvolver um sistema simples que:

  • Receba informações do usuário (nome, idade, notas, etc.)
  • Armazene esses dados em variáveis e listas
  • Utilize estruturas condicionais para decisões (ex: aprovação/reprovação)
  • Use funções para organizar o código
  • Salve e leia informações de arquivos
  • Opcional: implemente classes para objetos

Exemplo de estrutura do projeto

# Cadastro de alunos
class Aluno:
    def __init__(self, nome, notas):
        self.nome = nome
        self.notas = notas

    def media(self):
        return sum(self.notas)/len(self.notas)

# Criando alunos
aluno1 = Aluno("Hayan", [8,9,10])
mostrar aluno1.nome
mostrar aluno1.media()

Dicas para concluir o projeto

  • Divida o projeto em pequenas etapas.
  • Teste cada parte antes de avançar.
  • Use comentários para organizar o código.
  • Revise conceitos de variáveis, loops, funções e POO.

Parabéns! Ao finalizar este projeto, você terá consolidado todos os conhecimentos adquiridos no curso de programação e estará pronto para criar sistemas mais complexos.
